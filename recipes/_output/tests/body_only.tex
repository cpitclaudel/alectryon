
To compile:

\begin{quote}
\begin{alltt}
alectryon -{}-body-only body_only.rst -o body_only.html
    # reST → HTML; produces ‘body_only.html’
alectryon -{}-body-only -{}-backend latex body_only.rst -o body_only.tex
    # reST → LaTeX; produces ‘body_only.tex’
\end{alltt}
\end{quote}

\texttt{-{}-body-only} keeps only the document’s body (no \texttt{<html>}/\texttt{<head>} for HTML, no preamble or \texttt{\textbackslash{}begin\{document\}} for LaTeX):
