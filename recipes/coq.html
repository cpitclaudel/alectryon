This file shows how to process a LaTeX file with Alectryon. To compile it, use the following command:

      alectryon coq.html # HTML → HTML, produces ‘coq.annotated.html’

To use Alectryon with your own HTML file, just wrap your code in pre tags with the alectryon class. Here are two examples:

Goal True /\ True.
  split. (* .unfold *)
  - exact I.
    
  Check nat. (* .unfold *)
  - exact I.
Qed.
